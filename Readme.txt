        <h2>Overview</h2>
        <p>This project is a comprehensive web application featuring user authentication and a portfolio showcase. Developed using HTML, CSS, JavaScript, Bootstrap, and Firebase, it ensures a seamless and responsive user experience with a modern and attractive UI.</p>

        <h2>Features</h2>
        <ul>
            <li><strong>User Registration</strong>: A dedicated page for users to create an account.</li>
            <li><strong>User Login</strong>: Post-registration, users are redirected to a login page to access their accounts.</li>
            <li><strong>Portfolio Access</strong>: After successful login, users can view and manage their portfolio.</li>
            <li><strong>Responsive Design</strong>: Ensures optimal viewing across a wide range of devices (desktops, tablets, and mobile phones).</li>
            <li><strong>Firebase Integration</strong>: Secure authentication and data management using Firebase.</li>
            <li><strong>Elegant UI</strong>: Designed with Bootstrap for a clean, modern, and user-friendly interface.</li>
        </ul>

        <h2>Technologies Used</h2>
        <ul>
            <li><strong>HTML5</strong>: For structuring the web pages.</li>
            <li><strong>CSS3</strong>: For styling the web pages with custom designs.</li>
            <li><strong>JavaScript</strong>: For adding interactivity and handling user inputs.</li>
            <li><strong>Bootstrap</strong>: For responsive and attractive design elements.</li>
            <li><strong>Firebase</strong>: For user authentication and database management.</li>
        </ul>

        <h2>How It Works</h2>
        <ol>
            <li><strong>Create Account</strong>: Users sign up by providing their details on the registration page.</li>
            <li><strong>Login</strong>: Upon successful registration, users are prompted to log in.</li>
            <li><strong>Access Portfolio</strong>: After logging in, users can navigate to their portfolio page.</li>
        </ol>
